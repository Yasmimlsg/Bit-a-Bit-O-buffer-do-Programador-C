# Bit-a-Bit-O-buffer-do-Programador-C
Repositório dedicado ao armazenamento de exercícios, laboratórios e projetos desenvolvidos durante a graduação em Engenharia da Computação. O objetivo é documentar a evolução da lógica de programação e o domínio da linguagem C, explorando desde a sintaxe básica até o gerenciamento de memória de baixo nível.
